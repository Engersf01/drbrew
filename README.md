# dr.breW

> just what the doctor ordered

Landing page for **[drbrew.beer](https://drbrew.beer/)** — powered by DLAB.

## Stack

Plain static site (HTML + CSS), no build step. Deployed on [Vercel](https://vercel.com/).

## Develop locally

Open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Deploy

Pushes to `main` deploy automatically via Vercel's GitHub integration.

```bash
vercel --prod
```
